# SEO (Search Engine Optimization)
- `Search Engine Optimization`의 약자로 `검색 엔진 최적화`라는 뜻이다.

- 검색 엔진을 통해 웹사이트가 상위에 노출되도록 컨텐츠, 링크, 기술적 요소 등을 최적화하는 과정을 말한다.


<br />
<br />


## SEO의 중요성
- 대부분의 사용자들은 검색 결과의 첫 페이지에서만 정보를 찾고 검색을 이탈하는 경우가 많다.

- 검색 결과 상위에 노출되면 더 많은 사용자들이 방문할 수 있고, 잠재 고객의 증가로 이어질 수 있다.

- 광고 비용을 절감할 수 있다.


<br />
<br />


## 검색 엔진의 동작 원리
- 크롤링 (Crawling)
  - 검색 엔진 로봇(크롤러 또는 스파이더)이 웹을 탐색하며 웹페이지 내용을 수집한다.

- 인덱싱 (Indexing)
  - 크롤링으로 모은 정보를 분석하고 거대한 데이터베이스에 저장한다.

- 랭킹 (Ranking)
  - 사용자가 검색어를 입력하면 검색 엔진은 인덱싱된 정보 중 가장 관련성 높고 유용한 결과를 찾아 순위를 매긴다.

## SEO의 종류
- 온페이지 SEO (On-page SEO)
  - 웹사이트 내부의 컨텐츠와 HTML 소스 코드를 최적화하는 것이다.
      - `키워드 최적화` : 제목, 본문, 헤더 태그, 메타 태그에 적절한 키워드 삽입
      - `메타 태그` : 검색 결과에 표시될 제목(Title Tag)과 설명(Meta Description)을 최적화한다.
      - `URL 구조` : 간결하고 키워드가 포함된 URL을 사용한다.
      - `내부 링크` : 관련 콘텐츠끼리 연결하여 웹사이트의 네비게이션을 개선한다.
      - `이미지 최적화` : 파일 이름과 ALT 태그에 키워드를 포함한다.

- 오프페이지 SEO (Off-page SEO)
  - 웹사이트 외부에서 주로 백링크(Backlink)와 웹사이트의 신뢰도를 높이는 작업에 중점을 둡니다.
      - `백링크` : 다른 웹 사이트에서 자신의 사이트로 연결되는 링크가 있으면 좋다.
      - `소셜 미디어` : 소셜 미디어 플랫폼에서 콘텐츠를 공유하고 트래픽을 유도한다.
      - `리뷰와 평판 관리` : 고객 리뷰와 평가를 긍정적으로 관리.

- 기술적 SEO (Technical SEO)
  - 검색 엔진이 웹사이트의 컨텐츠를 효과적으로 크롤링하고 인덱싱할 수 있게 하는것에 중점을 둔다.
    - `웹사이트 속도` : 페이지 로딩 속도를 최적화한다.
    - `모바일 최적화` : 모바일 친화적인 디자인과 반응형 웹사이트로 제작한다.
    - `사이트맵` : 검색 엔진이 사이트 구조를 쉽게 이해할 수 있도록 XML 사이트맵을을 생성한다.
    - `로봇 파일(robots.txt)` : 검색 엔진이 크롤링 가능한 영역과 불가능한 영역을 지정한다.
    - `HTTPS 보안` : SSL 인증서를 통해 웹사이트를 안전하게 보호한다.
    - `중복 콘텐츠 관리` : 중복된 페이지를 정리하여 검색 엔진이 혼동하지 않도록 처리한다.

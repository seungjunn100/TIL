## 리액트 컴파일러

리액트 19와 함께 도입한 자동 성능 최적화 컴파일러로, 개발자가 별도의 `useMemo`, `useCallback`, `React.memo`를 직접 쓰지 않아도 코드를 분석하여 자동으로 메모이제이션과 렌더링 최적화를 수행한다.

<br />

### 설치 및 사용법

- 참고 사이트 : [https://ko.react.dev/learn/react-compiler](https://ko.react.dev/learn/react-compiler)

- 패키지 설치를 완료하면 `vite.config.ts`에서 플러그인까지 추가해주면 사용 가능한 상태가 된다.

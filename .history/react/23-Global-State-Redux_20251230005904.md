## Redux

- `Redux`는 `JavaScript` 애플리케이션의 상태(`state`)를 관리하기 위한 라이브러리다.

- 여러 컴포넌트가 공유하는 상태를 하나의 중앙 저장소(`Store`)에 두고 정해진 규칙으로 관리한다.

- `React` 자체는 `UI` 라이브러리고, `Redux`는 상태 변경을 규칙과 절차 안에서만 하도록 강제하는 설계도 같은 도구이다.

<br />

### Redux 특징

- 애플리케이션의 모든 상태를 하나의 저장소에서 관리한다.

- 상태 변경이 명확한 규칙을 따르므로, 버그를 찾고 디버깅하기 쉽다.

  - 상태 변경은 항상 순수 함수(`Reducer`)를 통해 이루어지며, 동일한 입력에는 동일한 출력이 나오므로 예측이 가능하다.

- 데이터는 컴포넌트(이벤트 발생) → 액션(작업 정의) → 리듀서(상태 관리) → 스토어(상태 저장) 순으로 한 방향으로 흐른다.
